Conclusions

When limitations become an advantage

Main lesson

Inheriting a legacy system is not a disadvantage, but an opportunity for evolutionary design

Maintaining the functionality of the backend while completely updating the interface became a key task, which led to the creation of a scalable system.

What worked

  • • Keep the backend running
  • • Create a system, not a set of screens
  • • Consider radically different scenarios
  • • Make the complex simple through personalization

What was difficult

  • • Balance between a unified style and platform features
  • • Transfer of the TV interface to mobile devices
  • • Train users on new features without pushing them away

For the future

Testing the effectiveness of color schemes
